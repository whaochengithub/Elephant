Form validation presented below uses jQuery Validation plugin. We have expanded this plugin so it can be used by writing HTML only.
To see real and more advanced examples please see Sign Up page.

In any form it is enough to set the attribute data-toggle="validator" and the rest will be taken care by the plugin itself. If a field has Help Text as in the following cases, the Help Text disappears to create space for the error message and reappears only when the field is successfully validated.
